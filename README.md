#🥘 Lasagna (JavaScript)
This repository contains my solution to the Lasagna exercise from Exercism.

📌 Description
The exercise simulates preparing and cooking a lasagna, focusing on calculating time-related values using simple JavaScript functions.

⚙️ Functionality
Calculates remaining oven time
Estimates preparation time based on layers
Computes total cooking time

🛠️ Tech
JavaScript (ES6)

🎯 Purpose
To practice writing clear, reusable functions and strengthen core JavaScript fundamentals.
